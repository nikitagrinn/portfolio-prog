    Решение задачи Two Sum¶

    Цель работы¶

    Разработать программу на языке Python для поиска двух элементов массива, сумма которых равна заданному числу. Также необходимо реализовать модульные тесты для проверки корректности работы программы.

    Постановка задачи¶

    Дан массив целых чисел nums и целое число target. Необходимо вернуть индексы двух чисел, сумма которых равна target.

    Условия задачи¶

    • существует не более одного решения;
    • один и тот же элемент нельзя использовать дважды;
    • если решения нет — вернуть пустой список.

    Примеры¶

    Пример 1¶

    Ввод: nums = [2, 7, 11, 15], target = 9
Вывод: [0, 1]

    Пример 2¶

    Ввод: nums = [3, 2, 4], target = 6
Вывод: [1, 2]

    Пример 3¶

    Ввод: nums = [3, 3], target = 6
Вывод: [0, 1]

    Описание алгоритма¶

    Для решения задачи используется метод на основе хэш-таблицы (словаря).

    Сложность¶

    Характеристика Значение
    Временная сложность O(n)
    Пространственная сложность O(n)

    По сравнению с методом полного перебора (O(n²)), данный алгоритм существенно эффективнее на больших массивах.

    Алгоритм работы¶

    1. Создать пустой словарь seen для хранения уже просмотренных элементов.
    2. Перебрать элементы массива с их индексами.
    3. Для каждого элемента вычислить diff = target - num.
    4. Если diff уже есть в словаре — найдена пара, вернуть [seen[diff], i].
    5. Иначе — добавить текущий элемент в словарь: seen[num] = i.
    6. Если перебор закончен и пара не найдена — вернуть [].

    Пример трассировки¶

    Для nums = [2, 7, 11, 15], target = 9:

    i num diff seen до Действие
    0 2 7 {} добавляем 2 → {2: 0}
    1 7 2 {2: 0} 2 найден → возврат [0, 1]

    Листинг программы¶

    Файл script.py¶

    import unittest

class Solution:
    def two_sum(self, nums: list[int], target: int) -> list[int]:
        seen = {}
        for i, num in enumerate(nums):
            diff = target - num
            if diff in seen:
                return [seen[diff], i]
            seen[num] = i
        return []

class TestTwoSum(unittest.TestCase):
    def setUp(self):
        self.solution = Solution()

    def test_basic_cases(self):
        self.assertEqual(self.solution.two_sum([2, 7, 11, 15], 9), [0, 1])
        self.assertEqual(self.solution.two_sum([3, 2, 4], 6), [1, 2])
        self.assertEqual(self.solution.two_sum([3, 3], 6), [0, 1])

    def test_custom_data(self):
        self.assertEqual(self.solution.two_sum([16, 2, 15, 89], 105), [0, 3])

    def test_negative_numbers(self):
        self.assertEqual(self.solution.two_sum([-1, -2, -3, -4, -5], -8), [2, 4])

    def test_zero_target(self):
        self.assertEqual(self.solution.two_sum([1, -1, 2, 3], 0), [0, 1])

    def test_no_solution(self):
        self.assertEqual(self.solution.two_sum([1, 2, 3, 4], 10), [])

    def test_duplicate_numbers(self):
        self.assertEqual(self.solution.two_sum([3, 3, 4, 5], 6), [0, 1])


if __name__ == '__main__':
    unittest.main()

    В ходе работы была реализована программа для поиска двух элементов массива, сумма которых равна заданному числу.

    Использован алгоритм на основе хэш-таблицы с временной сложностью O(n), что эффективнее классического перебора O(n²).

    Разработаны 6 модульных тестов, покрывающих базовые случаи, отрицательные числа, нулевой target, повторяющиеся элементы и отсутствие решения.
