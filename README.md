## Структура проекта

Проект построен с использованием генератора статических сайтов **MkDocs** (Material for MkDocs).

- `docs/` — исходные файлы в формате Markdown и скриншоты.
- `docs/programming/` — исходный код лабораторных работ (скрипты, тесты).
- `mkdocs.yml` — конфигурационный файл сайта.
